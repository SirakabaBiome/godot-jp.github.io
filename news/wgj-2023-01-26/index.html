"Godot 4: Baking lightmap for procedurally generated forest" を YouTube で見る

This example uses undocumented behaviour of Godot 4's LightmapGI node, which calls get_bake_bmeshes() to fetch meshes on each Node3D (excluding MeshInstance3D). Also I'm using patch which adds texel_scale to LightmapGI node, to shorten the baking time by reducing shadows texture size/quality: https://github.com/godotengine/godot/pull/64908
